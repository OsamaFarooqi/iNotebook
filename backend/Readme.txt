Started with
- npm init
- npm i express
- npm i mongoose (mongoose is a abstraction layer on top of MongoDB that supports in connecting with Node.Js app)
- npm i -D nodemon (here -D is to specifiy that add this as dev dependency)
- visit expressjs website and copy boiler plat code to connect with express
- npm install express-validator
- npm i bcrypt (to encript the password with hash and salt)
- npm i jsonwebtoken (to issue web tokens for authenticated users)
